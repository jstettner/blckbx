# Setup
Must have mongdb, express, node, npm.

`cd blckbx && yarn install`
`cd client && yarn install`

# Dev
- 1st tab
`cd blckbx && mongod`
- 2nd tab
`cd blckbx && yarn start`
- 3rd tab
`cd blckbx && gulp`
- 4th tab
`cd blckbx/client && yarn start`

# Build
`cd blckbx/client`
`gulp sass && yarn build`
